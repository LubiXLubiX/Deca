# LubiX Software (CLI)

This folder contains the standalone **LubiX Software CLI**, intended to be installed globally (like Flutter).

## Development

```bash
go build -o lubix
./lubix version
./lubix run dev
```

## Commands (planned)

- `lubix version` / `lubix -v`
- `lubix create-project <name>`
- `lubix run dev`
- `lubix migrate`
